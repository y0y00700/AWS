1장. 도메인 연결하기. Route 53
Route 53 : 도메인을 발급하고 관리해주는 서비스이다.
Route 53은 DNS 서비스이다.

※ DNS (Domain Name System) 
숫자로 된 IP 주소를 문자로 연결가능하게 만드는 시스템.


2장.Route 53에서 도메인 구매
AWS > Route53 검색 > 도메인 > 도메인 등록 > 도메인 검색

레코드 유형
A ,  CNAME 형태가 주로 쓰임.

빠른 레코드 생성  > 값  : 할당 받은 IP 주소  입력

레코드 이름 : 서브 도메인 설정인듯 하다..

보충.
무료로 도메인 구매하는 방법.

내도메인.한국 검색
발급 받은 후, IP 주소와 매핑해주면  도메인 적용이 가능하다.
